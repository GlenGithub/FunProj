# FunProj
一个简单的图片视频音乐app,material design风格。项目结构采用mvp。retrofit2封装,recycleview加载,mediaPlay管理类的封装，简单的音乐播放,glide图片加载。

# api
数据来自百度、美拍、网易云音乐的开放接口。
